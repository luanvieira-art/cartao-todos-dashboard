Importar Planilha Excel

Siga este formato para que a importação funcione corretamente.

  • A aba de dados deve começar com a palavra "DADOS" (ex.: DADOS MAIO.26). Pode haver mais de uma aba, uma por mês.
  • A Linha 1 é um título livre, a Linha 2 são os cabeçalhos das colunas e os dados começam na Linha 3.
  • Não altere a ordem nem o nome das colunas — eles precisam ser idênticos ao modelo.
  • CONVERSÃO aceita: SIM, NÃO, INCONFORMIDADE, 7 DIAS, ÓBITO, EM TRATATIVA ou OUVIDORIA.
  • TIPO DE RETENÇÃO aceita: UNIDADE ou TELEFONE.
